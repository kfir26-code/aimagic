# AI Magic — Landing Page

Marketing landing page for **AI Magic** (websites, automations & AI for businesses).

- **Live site:** https://kfirperez.com
- **Host:** Vercel — project `ai-magic`
- **Deploys:** automatically on every push to `main`

## Files

| File | Purpose |
|------|---------|
| `index.html` | The landing page (single-file static site, React via CDN) |
| `tweaks-panel.jsx` | Design-tweak panel — invisible to visitors, required by the page |

## Updating the site

Edit the files, then:

```sh
git add .
git commit -m "describe your change"
git push
```

Vercel builds and publishes the change within seconds.
